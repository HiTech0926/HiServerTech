
# HiServerTech


## 关于

这是一个Hi世界服务器使用的粘液附属插件，使用MIT许可证开源，欢迎各位提出好的想法意见！
意见提出/Bug反馈：<[https://github.com/HiTech0926/HiServerTech/issues](https://github.com/HiTech0926/HiServerTech/issues)>
